05 — safety guardrails
🛡️
kube-system Protected
The kube-system namespace is hardcoded as off-limits. No experiment can target it.
🎯
Blast Radius Limit
Default max 30% of matching pods. Configurable but never more than you explicitly set.
🔁
Auto Rollback
Every workflow has an always: cleanup block. Chaos artefacts are removed on success or failure.
🔵
Dry Run Default
Dry run is ON by default. Every experiment shows exactly what it will do before it does it.
📋
Full Audit Trail
Every run is logged in GitHub Actions history. JSON reports saved as artifacts for 30 days.
⚠
Emergency Rollback
One-click rollback button cleans all chaos artefacts, uncordons nodes, and restarts deployments.
